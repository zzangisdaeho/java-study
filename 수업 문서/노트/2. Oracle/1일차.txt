Database : 한 조직에 필요한 정보를 여러 응용 시스템에서 공용할 수 있도록
	논리적으로 연관된 데이터를 모으고 중복되는 데이터를 최소화하여 (DB 모델링)
	구조적으로 통합/저장해놓은 것
	그러니까 DB는 데이터를 정렬하여 저장해놓는 곳, 정보는 DB 데이터 이용해서 알아서 이용들하셍

데이터베이스는 자바와 다르게 메모리 주소에 집중하는게 아니고, 데이터 자체의 값에 초점을 맞춘다.

DBMS : DB에 접근하여 조작하는 도구

관리자계정(SYSTEM계정)의 사용자 이름은
★ sys as sysdba ★
로 고정되어있다. 그냥 그렇게 만들었덴다



테이블 : 데이터들이 들어가있는 공간

공용 동의어 : Data Dictionary(DD) : 데이터 사전. 테이블 외의 모든 정보가 다 들어간다.
