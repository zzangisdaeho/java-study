변수


변수 : 데이터를 저장하는 공간

기본자료형 : 8개 (이름과 크기)

변수명 명명 규칙(대소문자 구분, 길이제한X, &_ 특수만자만 사용 가능, 낙타봉 표기법, 예약어사용금지)

변수는 stack에 생성됨

상수 = final

오버플로우 = 값 초과시 맨처음으로 감

형변환 = 강제형변환, 자동형변환 있싐



연산자

전위연산자 : 내가먼저 ++a, 후위연산자 : 다른애먼저 a++

% : 나머지 구하는것
/ : double로 나눌시 double값

3항 연산식 : (조건식)? 결과1 : 결과2

= 과 ==의 차이 : = 대입 == 같다 
문자열끼리 ==는 주소값 비교 값비교는 equals


제어문

조건문, 반복문 연습



배열

배열 : 같은 자료형을 여러개 묶음으로 묶어놓은 것

2차 배열 : for문 두개 써야 관리 가능


객체

클래스에 정의되어 있는 것을 new연산자와 생성자를 통해 heap영역에 올리는것

클래스는 객체를 만들기 위한 도안


클래스 안에 : 필드, 생성자, 메서드
	★클래스 : 생성제한자 퍼블릭, 디폴트만 가능

	필드 : 클래스 영역에 선언된 것 접근제한자 예약어 클래스 클래스명

	생성자 :  객체를 생성해주며 필드를 초기화함. 클래스이름과 같음. 반환값없음.

	메소드 : 리턴값과 매개변수에 따른 4가지 타입

접근제한자 4가지 각각 범위


파이널 붙을 수 있는 곳 : 클래스(상속불가) 필드(상수) 메서드(오버라이딩 불가) 지역변수(상수)
