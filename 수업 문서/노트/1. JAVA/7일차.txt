do{
}while;

 do{} 단 안에서 선언한 변수(자료형 포함)은 지역변수로 do단 안에서만 사용 가능
while단에서는 사용할 수 없다.

지역변수 개념에 관하여

지역변수는 함수 안에서 혹은 { } 안에서만 사용 가능한 함수를 말한다.
이 중 { } 안에서만 사용 가능한 함수는 다음과 같은 성질을 갖는다.

지역변수는 사용된 수행문이 있는 { } 영역 안에서만 움직인다.
위의 do{ } while에서 do  안에 선언한 변수가 while에서 쓸 수 없는것도 do에서 } 가 닫혔기 때문

중첩구문에선 다음과 같다

for{ 1번 변수 선언
 for{ 2번 변수 선언
  }
 } 

1번 변수는 2번 변수 안에서 사용 가능 (큰 {   } 구문이 맨 마지막에 }로 닫히니까)
2번 변수는 1번 변수 안에서 사용 불가능 (작은 {  } 구문이 큰 구문 전에 닫히니까)
포괄 형태에서 변수가 바깥으로 나갈순 없지만 안쪽에 있는건 바깥이 { }로 감싸고 있기 때문에 사용 가능하다.
